About Foundation

Foundation is a type of front-end framework. You can work with HTML, CSS, or JavaScript. Foundation provides responsive UI componenets, templates, and code snippets. We will be showcasing some of the cooler features of Foundation that are completely JS free!

Callouts

Callouts combine panels and alerts into one generic container component.

This is a callout!

A callout is an element with the .callout class applied. You can put any kind of content inside.
You can create a callout with the following code:
<div class = "callout">
<p> content </p>
</div>

This is another callout, but secondary!

Callouts can be colored using the .secondary, .primary, .success, .warning, or .alert classes.
For example, this callout uses <div class = "callout secondary">

This is a callout, but with a warning!

You can also make a callout closable by adding a closable button and the data-closable attribute. This creates a dismissable alert box like this one if motion-UI is added. Clicking on the 'X' won't do anything... for now.

XY Grid

The XY Grid has multiple grid types, like margin grid, padding grid, frame grid, block grid, and vertical grid. It works similarly to the standard float grid, but includes a number of useful features like horizontal and vertical alignment and autosizing.
