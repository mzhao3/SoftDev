Breaking it down

With Bootstrap, you can put your content in containers , which is required when using the defualt grid system. This content is in a container, which centers and horizontally pads our content. It also resizes whenever you change the size of the screen. Replicate it with the following code: 


    <div class = "container">
    ...
    </div>

You can also remove the padding by making the container full width with this code: 


    <div class = "container-fluid">
    ...
    </div>

Buttons

Create these super clickable buttons quickly and easily.


Button code: 

    <button type="button" class="btn btn-name">name</button>

You can also create buttons without the color in the background. Simply replace the default modifier class with .btn-outline-* to remove all images and colors on a button like so:


Outlined buttons: 

    <button type="button" class="btn btn-outline-name">name</button>

Alerts

Alerts provide feedback messages for typical user actions and can be used for any length of text. They are created the same way that buttons are- use one of the eight required classes. .alert-primary, .alert-secondary, .alert-success, .alert-danger, .alert-warning, .alert-info, .alert-dark or .alert-light.

Tables

It's also easy to create tables with a variety of stylings. Tables in Bootstrap 4 are inherited, so nested tables will be styled in the same manner as the parent. Simply add the base class .table to a <table> and extend with custom styles.

# First Last Color
1 Jimmy James Raspberry
2 Mark Zuccerberg Electric Blue
3 Donald Trump Cheeto Orange

This table: 

    <table class="table">
      <thead>
        <tr>
          <th scope="col">#</th>
          <th scope="col">First</th>
          <th scope="col">Last</th>
          <th scope="col">Color</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <th scope="row">1</th>
          <td>Jimmy</td>
          <td>James</td>
          <td>Raspberry</td>
        </tr>
        <tr>
          <th scope="row">2</th>
          <td>Mark</td>
          <td>Zuccerberg</td>
          <td>Electric Blue</td>
        </tr>
        <tr>
          <th scope="row">3</th>
          <td>Donald</td>
          <td>Trump</td>
          <td>Cheeto Orange</td>
        </tr>
      </tbody>
    </table>

You can also invert the colors (light text on dark background) with .table-dark for white text on a black table, .thead-light for black text on a light grey header, or .thead-dark for white text on a black header.
